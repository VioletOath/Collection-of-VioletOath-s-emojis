## ✨ 项目简介

这是一个包含 **478+** 个精选中文表情包的开源资源库。所有表情包采用 PNG 格式，支持透明背景，使用直观的中文命名，无需分类目录，所有文件平铺存放，方便快速查找和使用。无论是日常聊天、社交媒体还是工作沟通，都能找到合适的表情包来表达你的心情。

## 📂 文件结构

```
500 个表情包/
├── *.png            # 478+ 个表情包文件（中文命名）
├── README.md        # 项目说明文档
└── LICENSE          # MIT 许可证
```

> 💡 **设计特点**：所有表情包文件直接存放在根目录，采用中文命名，无需进入多层目录即可找到所需表情包，支持系统搜索快速定位。

## 📖 使用方法

### 1. 本地使用

```bash
# 克隆整个仓库到本地
git clone https://github.com/VioletOath/500 个表情包.git
cd 500 个表情包
```

### 2. 在聊天工具中使用

- **微信/QQ**: 直接将 PNG 文件拖入聊天窗口
- **钉钉/企业微信**: 点击表情图标 → 选择本地图片
- **Discord/Slack**: 上传图片或拖拽发送

### 3. 在网页中使用

```html
<!-- 本地引用 -->
<img src="./开心.png" alt="开心" width="100">

<!-- GitHub CDN 引用（示例） -->
<img src="https://raw.githubusercontent.com/VioletOath/500 个表情包/main/开心.png" alt="开心">
```

### 4. 在 Markdown 文档中使用

```markdown
![开心](./开心.png)
![加油](./fighting.png)
```

### 5. 快速查找技巧

- **Windows**: 在文件夹中按 `Ctrl+F` 搜索中文关键词
- **macOS**: 使用 Spotlight 或 Finder 搜索功能
- **Linux**: 使用 `ls | grep 关键词` 命令
